# Monocle is a fully open source view-extension for the [Dynamo Visual Programming environment](https://dynamobim.org/).

## License

## Available Tools
- Package Usage
- Custom Node Highlighting
- FOCA
  - Alignment
  - Grouping
  - Pow List
  - Combinifier
  - Dropdown Converter 3000
- Custom Settings
  - Group Settings
  - Deployment
- Simple Search
- Standard Views
- UI Conformity for Revit
